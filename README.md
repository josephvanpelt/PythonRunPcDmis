# PythonRunPcDmis
Python run PC-DMIS is a simple script that demonstrates the use of automation to drive PC-DMIS from a python script.

## Requirements
* Installation of PC-DMIS
* Python 2.7 or 3.7 for Windows
* (pip install pywin32)
